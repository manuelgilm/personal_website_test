# 🎬 Welcome to YouWiki

**Documentation companion for video tutorials across multiple projects**

YouWiki is a comprehensive documentation platform designed to complement video content from across my projects. Rather than transcripts or scripts, you'll find practical, structured guides that let you follow along with videos, copy-paste code examples, and reference key concepts without constantly rewinding.

## 🎯 What is YouWiki?

This site serves as a central hub for documentation, tutorials, and guides covering various topics and projects. Whether you're learning a new framework, setting up infrastructure, or building a project from scratch, you'll find:

- **Step-by-step guides** to follow alongside video tutorials
- **Code snippets & examples** ready to use in your projects
- **Architectural overviews** explaining design decisions
- **Configuration references** and troubleshooting tips
- **Best practices & lessons learned** from real-world implementations

## 📂 Available Channels

Explore different areas of focus by browsing the channels in the navigation menu:

## 💡 How to Use This Site

1. **Pick a channel** - Browse the navigation to find the topic area you're interested in
2. **Watch the video** - Follow along with the corresponding YouTube tutorial
3. **Reference the docs** - Use this site as a companion to copy code, check configurations, and understand key concepts
4. **Learn at your pace** - Revisit sections anytime for reference or deeper understanding

## 🔍 Finding What You Need

- **Top navigation** - Select a channel to explore
- **Left sidebar** - Browse specific tutorials and topics within each channel
- **Search** - Use the search functionality to find topics across all content
